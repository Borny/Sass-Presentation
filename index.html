Responsive Grid

Basée sur 12 colonnes

Modal

Utilise un input checkbox

Par défault la checkbox est unchecked, lorsqu'elle est cochée la modal est activée et apparaît à l'écran.

Pour la faire disparaître il suffit de clicker sur un bouton ou de clicker n'importe où en dehors de la modal. Le bouton ainsi que l'extérieur de la modal utilise des balises

Du JavaScript est utilisé pour ajouter un overflow:hidden à l'élément afin de désactiver le scroll dans la page.

Collapse

Utilise un input checkbox

Par défault la checkbox est unchecked, lorsqu'elle est cochée l'élément se décollapse.

Malheureusement pas possible de faire interragir les éléments collapse entre eux sans JavaScript

Tab

Utilise un input radio

Par défault l'input radio est unchecked, lorsqu'il est coché l'élément tab apparaît.

L'input radio du premier élément est coché par défault.
