Nesting

Formation Sass

Imbriquation des sélecteurs

Permet d'imbriquer les media queries

Pas plus de 4 niveaux

// Code CSS

body{ height: 100%; background: $white; } body .header{ width: 100%; padding: 15px; } body .header .content{ font-size: 3em; color: $gray; font-weight: bolder; }

// Code Imbriqué

body{ height: 100%; background: $white; .header{ width: 100%; padding: 15px; .content{ font-size: 3em; color: $gray; font-weight: bolder; } } }

$variables

Formation Sass

Permet de stocker une valeur et de l'appeler n'importe où

Besoin de modifier la valeur à un seul endroit

$blue: #A3EAB;
$font-size: 2em; body{ height: 100%; background: $blue; .header{ width: 100%; padding: 15px; .content{ font-size: 3em; color: $blue; font-size: $font-size; } } }

@mixins

Formation Sass

Permet de rassembler un ensemble de règles réutilisables

Peut utiliser des paramètres

Permet d'avoir moins de code à l'écran

Créer avec @mixin

Appeler avec @include

// Création de la mixin

@mixin inlineBlock($position){ display: inline-block; vertical-align: $position; }

// Appel de la mixin

button{ @include inlineBlock(middle); }

%placeholders

Formation Sass

Permet de rassembler un ensemble de règles dont les valeurs ne varient pas

Créer avec % et le nom du placeholder

Appeler avec @extend

// Création du placeholder

%transition { -webkit-transition: 0.4s; -moz-transition: 0.4s; -ms-transition: 0.4s; -o-transition: 0.4s; transition: 0.4s; -webkit-animation-timing-function: linear; -moz-animation-timing-function: linear; -ms-animation-timing-function: linear; -o-animation-timing-function: linear; animation-timing-function: linear; }

// Appel du placeholder

button{ @extend %transition; }

@extend

Formation Sass

Permet de dupliquer des règles existantes

Créer avec @extend et le nom du sélecteurs

// SCSS

header{ height: 100%; background: $blue; } footer{ @extend header; width: 100%; padding: 15px; }

// CSS

header{ height: 100%; background: $blue; } footer{ height: 100%; background: $blue; width: 100%; padding: 15px; }

Modal

Formation Sass

Utilise un input checkbox

Par défault la checkbox est unchecked, lorsqu'elle est cochée la modal est activée et apparaît à l'écran.

Pour la faire disparaître il suffit de clicker sur un bouton ou de clicker n'importe où en dehors de la modal. Le bouton ainsi que l'extérieur de la modal utilise des balises <label> pour décocher la checkbox.

Du JavaScript est utilisé pour ajouter un overflow:hidden à l'élément <body> afin de désactiver le scroll dans la page.
